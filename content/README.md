# Essays / Blog Posts

This directory contains essay and blog post content in Markdown format.

## File Naming Convention

Use the format: `YYYY-MM-DD-slug-title.md`

Example: `2025-11-22-my-first-essay.md`

## Essay Format

Each essay should have frontmatter at the top:

```markdown
---
title: "Your Essay Title"
date: 2025-11-22
author: BirdyPhillips
tags: [tag1, tag2]
published: true
---

Your essay content here...
```

## Organization

- Published essays go directly in `essays/`
- Drafts can go in `essays/drafts/` (optional)
- Images for essays can go in `app/static/blog-images/`
